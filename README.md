# libkripto
*Simple and flexible cryptographic library written in ISO C99.*

**Due to lack of time and interest project is abandoned.**

**IT IS NOT TESTED**

There is no documentation, however API is really simple.

Run build.sh to compile.

* ARIA

* CAST5
* Camellia
* DES (TDES)
* GOST
* IDEA
* KHAZAD

* Skipjack

* Speck32
* Speck64

* Threefish512
